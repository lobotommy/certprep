W3C model

element.addEventListener('event', function, bool); // the bool determines if the handler should kick in when the event is captured (true) or bubbles (false).
In the lastest version, not yet in browsers, there is a way to find all registered eventhandlers for an event through

Microsoft-model

element.attachEvent('event', function);
Has two drawbacks, elements always bubble and the event handling function is referenced, not copied. This leads to the fact that 'this' is not supported.
